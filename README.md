# Capacity Planning Models — Next.js App

This UI allows users to explore mathematical models for network and system capacity planning, including Erlang-B. Enter system parameters, calculate call blocking probabilities, and view step-by-step calculations.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
